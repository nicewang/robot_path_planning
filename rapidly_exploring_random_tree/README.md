### Appendix: Build & Run
`CMakeList.txt` mainly for github workflows' building process.

If you want to build manually, use `Makefile` there. 

* Build

```shell
make
```

* Run:

```shell
./rrt.so
```

* Clean:

```shell
make clean
```
